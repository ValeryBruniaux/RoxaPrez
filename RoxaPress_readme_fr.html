# Avantages de RoxaPress --- #### Pas de perte de temps avec la mise en pages #### Rédaction du contenu facile (Mardown) #### Un seul fichier (texte) à éditer #### Toutes les possibilités de style du CSS #### Fonctionne avec un simple navigateur internet #### Publication en ligne sans aucune modification
# Utilisation de RoxaPress --- #### Copiez le répertoire RoxaPress #### Renommez le avec le nom de votre présentation #### Dans le répertoire, éditez le fichier index.html #### Modifiez le style en éditant style.css #### Visualisez avec votre navigateur
